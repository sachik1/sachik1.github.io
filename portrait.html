Self Portrait

This self-portrait is a creative visual interpretation of myself, made using drawing primitives in p5.js. I used shapes like ellipses, arcs, and rectangles to sketch out my likeness â€” focusing on features like hair, glasses, and sweater vest.

Code Used:

function setup() {
  createCanvas(600, 600);
  background(255, 245, 250);
  noStroke();

  fill(40, 30, 20);
  ellipse(300, 320, 300, 400);
  ellipse(190, 380, 90, 190);
  ellipse(410, 380, 90, 190);

  fill(255, 220, 200);
  rect(280, 390, 40, 80, 20);
  ellipse(300, 280, 200, 240);

  fill(255, 160, 160, 90);
  ellipse(250, 320, 30, 15);
  ellipse(350, 320, 30, 15);

  fill(40, 20, 10);
  ellipse(260, 270, 20, 26);
  ellipse(340, 270, 20, 26);

  fill(255);
  ellipse(263, 265, 5, 5);
  ellipse(343, 265, 5, 5);

  fill(40, 30, 20);
  rect(240, 245, 40, 5, 2);
  rect(320, 245, 40, 5, 2);

  fill(255, 210, 190);
  ellipse(300, 285, 8, 8);

  fill(230, 100, 130);
  arc(300, 310, 50, 30, 0, PI);

  stroke(90);
  strokeWeight(2);
  noFill();
  rect(230, 250, 55, 35, 10);
  rect(315, 250, 55, 35, 10);
  line(285, 265, 315, 265);
  noStroke();

  fill(255);
  rect(220, 400, 160, 140, 20);

  fill(130, 200, 140);
